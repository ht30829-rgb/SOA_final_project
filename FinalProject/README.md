# FinalProject

# Before opening this project, ensure you have the following installed on your machine:

# Visual Studio (latest version)
# SQL Server Management Studio (SSMS) (for working with the database)
# .NET SDK (appropriate version for the project)

# Steps to Open the Project

# Clone the Project
# Open the Project in Visual Studio (.csproj)
# Set Up the Database in SQL Server Management Studio (SSMS)
# Run migration to set up the Database schema
# Build and Run the Project
# Test the Application
